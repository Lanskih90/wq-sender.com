# Зображення лендінгу

`favicon.svg` та `og-image.svg` уже згенеровано.

## Скріншоти панелі (додайте самостійно)

Секція «Демо панелі» на головній очікує 5 файлів. Збережіть сюди ваші скріншоти
з такими іменами (PNG або JPG; рекомендований формат ~16:10, ширина 1200–1600px):

| Файл                      | Що на скріні            |
|---------------------------|-------------------------|
| `screen-chats.png`        | Чати                    |
| `screen-connections.png`  | Підключення             |
| `screen-scenarios.png`    | Сценарії                |
| `screen-users.png`        | Користувачі             |
| `screen-broadcast.png`    | Розсилання              |

Поки файлів немає — на їхньому місці акуратно показується стилізована заглушка
(обробляється у `assets/js/main.js`), тож сайт не виглядає «зламаним».

## OG-зображення

`og-image.svg` готове. Деякі соцмережі краще показують прев'ю з PNG —
за бажання експортуйте `og-image.svg` у `og-image.png` (1200×630) і змініть
`og:image` у `index.html` на `.png`.
